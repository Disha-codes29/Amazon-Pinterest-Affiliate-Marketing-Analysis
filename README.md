# Amazon Affiliate Marketing on Pinterest - India Trend Analysis 2021-2026

## Project Overview
This is an end to end data analytics project analyzing Amazon Affiliate 
Marketing trends on Pinterest in India using real Google Trends data.

## Tools Used
- Python (Google Colab)
- Pandas and NumPy
- Pytrends API
- Scikit-learn (Random Forest)
- Matplotlib and Seaborn
- Power BI

## Project Phases
- Phase 1: Data Collection using Pytrends API
- Phase 2: Data Cleaning with Pandas and NumPy
- Phase 3: Trend Analysis and Sentiment Analysis
- Phase 4: Machine Learning Model (98.08% accuracy)
- Phase 5: Power BI Dashboard

## Key Findings
- Affiliate Marketing is strongest keyword in India (score 44.2)
- Pinterest Affiliate Marketing is untapped in India
- ML model predicts steady growth April to July 2026
- Peak interest was recorded in early 2022
